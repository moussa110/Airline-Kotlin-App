# Airline App
android application.

An Application that helps the user to explore different airlines, show airline details and filter airlines by (name, country, id).
Using:
- kotlin.
- Mvvm+liveData.
- Cashing with room.
- dagger hilt.
- retrofit.
- Kotlin coroutines.
- Unit testing.
- nav graph.
